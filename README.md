# Cloud Based File storage system -AWS S3
## Project description 
This project demonstrate a cloud based file storage system using AWS S3.
The Implementation is done using AWS lab environment without a personal aws account.

## Features
File upload to s3 bucket 
File download from s3 bucket 
Secure cloud storage using aws s3
Practical hands on lab implementation 

## Tools and Technology 
aws s3
aws lab environment 
github

## Outcome 
Successfully created and managed an s3 bucket, uploaded and accessed files using aws cloud services
